Assignment prompt:
Make a hybrid program that complies with the following structural diagram

This style of programming is sometimes called hybrid programming because it uses functions taken directly from the libraries of C and C++. 
Sometime there is no known function in the C++ library of functions that meets our needs, then we create a brand new C++ function and use it when needed.
